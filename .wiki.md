# Project Summary
UAHAB is a modular digital platform aimed at supporting the Ukrainian community in Spain. It fosters connections, resource sharing, and access to vital services through a comprehensive ecosystem that includes job listings, housing, services, a marketplace, community events, and a business directory. The platform enhances user experience by celebrating Ukrainian culture while providing essential resources for integration and support in Spain. Recent updates have introduced a real-time messaging system to facilitate communication between users.

# Project Module Description
The project consists of a frontend and backend architecture with various functional modules, including:
- **Jobs Module**: Structured job listings for vacancies, job seekers, and employment assistance.
- **Housing Module**: Listings for available housing, including rentals and sales.
- **Services Module**: A directory for various services such as legal, educational, and household help.
- **Marketplace Module**: A space for peer-to-peer and semi-commercial transactions.
- **Events Module**: Listings for community and business events.
- **Community Module**: A platform for social connections and community initiatives.
- **Organizations Module**: Information about associations, resources, and support services.
- **Business Directory Module**: A directory for Ukrainian businesses in Spain.
- **Messaging Module**: A real-time messaging system allowing users to communicate about listings, including inbox management, notifications, and conversation threads.

# File Description Inventory
- **backend/**: Contains backend logic, data models, and database migrations.
- **frontend/**: Contains the user interface, including components, pages, and assets.
- **README.md**: Documentation for both backend and frontend, explaining setup and usage.
- **package.json**: Lists dependencies and scripts for the frontend.
- **pnpm-lock.yaml**: Lockfile for frontend dependencies.
- **src/**: Main source code for the frontend, including pages and components.
- **lib/**: Contains utility functions and context providers for internationalization and theming.
- **messaging.py**: Backend API for the messaging system, handling sending messages, fetching inbox, and managing conversation threads.
- **MessagesPage.tsx**: Frontend component for displaying and managing user messages.
- **AdBanner.tsx**: Component for displaying advertisements throughout the platform.
- **SearchPage.tsx**: Component for handling global search functionality across listings and businesses.

# Technology Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS
- **Backend**: Python, FastAPI, Alembic for migrations, Atoms Cloud for data persistence
- **Database**: Managed through Alembic migrations

# Usage
1. **Install Dependencies**:
   - For backend: Navigate to `app/backend` and run `pip install -r requirements.txt`.
   - For frontend: Navigate to `app/frontend` and run `pnpm install`.

2. **Build the Project**:
   - For backend: Run the necessary commands to set up the database and migrations.
   - For frontend: Run `pnpm run build` to compile the assets.

3. **Run the Project**:
   - Start the backend service and frontend application according to the respective instructions in their README files.
